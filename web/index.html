How to use:

1. Load mp3 of your choice

2: Hold down ctrl and click n' drag to create regions

3: Double click on region to rename label and add description

4: Press enter in each text field when edits are done

5: Annotation can be saved to a JSON-file if save icon is pressed

6: Saved JSON can be reloaded with the 'Load JSON btn'

Hold down ctrl to enable region creation!
